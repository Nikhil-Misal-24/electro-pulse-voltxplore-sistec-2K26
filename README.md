# ⚡ VOLTXPLORE | 19th Sagar Fiesta

Welcome to the official repository of **VoltXplore**, a premier electronics-based technical competition organized by the **Electro Pulse Club (ECE Department)** at **Sagar Institute of Science & Technology (SISTec), Bhopal**.

## 🚀 About the Event
VoltXplore is designed to test the practical skills, technical logic, and debugging capabilities of future engineers. Part of the grand **19th Sagar Fiesta**, this event brings together electronics enthusiasts for three thrilling rounds:

1. **Visual Instinct:** Component identification from macro-photography and hardware sounds.
2. **The Tech-Trail:** A department-wide technical scavenger hunt.
3. **The Debugging Challenge:** Restoring functionality to malfunctioning circuits.

## 🛠️ Built With
* **HTML5** - Structure of the web portal.
* **CSS3** - Custom Neon-themed UI and Glassmorphism design.
* **JavaScript** - AOS (Animate On Scroll) and interactive elements.
* **Formspree** - Integration for real-time registration data.
